## 🎯 О проекте

GiftGenius - это интеллектуальный помощник, который помогает подобрать идеальный подарок на основе описания человека. Вместо простого поиска по ключевым словам, система:
1. Анализирует психологический портрет получателя
2. Определяет, что НЕЛЬЗЯ дарить
3. Генерирует уникальные персонализированные идеи
4. Предоставляет ссылки на поиск этих подарков в популярных маркетплейсах

## 🏗 Архитектура

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   Streamlit     │────▶│   FastAPI        │────▶│   Groq API      │
│   (app.py)      │     │   (api.py)       │     │   (LLM)         │
└─────────────────┘     └──────────────────┘     └─────────────────┘
         ▲                       │
         │                  ┌────▼──────┐
         │                  │  RAG      │
         └──────────────────│  Pipeline │
         (локальный         └────┬──────┘
          интерфейс)        ┌─────┴──────┐
                           ▼             ▼
                   ┌──────────────┐ ┌──────────────┐
                   │  FAISS Index │ │  BM25 Index  │
                   │ (эмбеддинги) │ │ (ключевые    │
                   └──────────────┘ │    слова)    │
                                    └──────────────┘
```

## 🔄 Как это работает

1. **Поиск** - По запросу ищутся релевантные товары в базе (гибридный поиск: FAISS + BM25)
2. **Портрет** - Groq API создает психологический портрет получателя
3. **Анти-предпочтения** - Анализируется, что НЕЛЬЗЯ дарить
4. **Генерация** - Groq API генерирует 5 уникальных идей подарков
5. **Ссылки** - Для каждой идеи создаются ссылки на поиск в маркетплейсах

## 🚀 Запуск в Kaggle

### 1. Создание ноутбука
1. Зайдите на [kaggle.com](https://kaggle.com)
2. Создайте новый ноутбук: **File → New Notebook**
3. Включите GPU: **Settings → Accelerator → NVIDIA T4**

### 2. Загрузка датасета
1. В правой панели нажмите **Add Data**
2. Загрузите датасет gifts_dataset_ru_plus (1).csv
3. Дождитесь загрузки
4. Добавьте путь к датасету в код

### 3. Настройка переменных
В первой ячейке ноутбука заполните свои данные:

```python
# ========== НАСТРОЙКИ ==========
NGROK_AUTH_TOKEN = "ваш_токен_ngrok"           # https://dashboard.ngrok.com
GROQ_API_KEY = 'ваш_ключ_groq'                 # https://console.groq.com
DATA_PATH = "/kaggle/input/datasets/..."       # путь из Add Data
```

### 4. Запуск
Выполните все ячейки по порядку. После запуска вы увидите:
```
🌐 URL: https://your-username.ngrok-free.dev
📌 POST https://your-username.ngrok-free.dev/generate
```

💻 Запуск локально
1. Установка Python
Скачайте Python 3.11 (рекомендуется) или 3.10-3.12 с python.org

⚠️ Важно: Проект тестировался на Python 3.11. Более новые версии (3.12+) могут иметь проблемы с некоторыми зависимостями. Python 3.9 и ниже не поддерживаются.

Проверьте версию Python:

bash
python --version
# Должно быть: Python 3.11.x
2. Клонирование репозитория
bash
git clone https://github.com/yourusername/gift-genius.git
cd gift-genius
3. Создание виртуального окружения
bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
4. Установка зависимостей
Создайте файл requirements.txt:

txt
streamlit==1.28.0
requests==2.31.0
urllib3==2.0.7
Установите зависимости:

bash
pip install -r requirements.txt
5. Настройка app.py
Скопируйте URL из вывода Kaggle и вставьте в app.py:

python
API_URL = "https://your-username.ngrok-free.dev/generate"
HEALTH_URL = "https://your-username.ngrok-free.dev/health"
6. Запуск Streamlit
bash
streamlit run app.py
Приложение откроется в браузере по адресу http://localhost:8501

## 🔑 Получение токенов

2. Перейдите в раздел **API Keys**
3. Нажмите **Create API Key**

## 📡 API Endpoints

| Endpoint | Метод | Описание | Пример ответа |
|----------|-------|----------|---------------|
| `/` | GET | Информация о сервере | `{"message": "🎁 Gift Genius v5 + Groq", "status": "healthy"}` |
| `/health` | GET | Проверка работоспособности | `{"status": "healthy", "timestamp": 1234567890}` |
| `/generate` | POST | Генерация идей подарков | `{"ideas": [...], "processing_time": 5.67}` |

### Пример запроса к API
```python
import requests

API_URL = "https://your-username.ngrok-free.dev/generate"

response = requests.post(
    API_URL,
    json={"description": "маме 50 лет, любит готовить и читать"},
    timeout=30
)

print(response.json())
```

## ❗ Устранение проблем

### Ошибка 404 в Streamlit
```
❌ Ошибка: 404 Client Error: Not Found for url
```
**Решение:** Проверьте URL в `app.py` - он должен совпадать с URL из Kaggle и заканчиваться на `/generate`

### Сервер не запускается в Kaggle
```
ERROR: [Errno 98] Address already in use
```
**Решение:** Перезапустите ядро:
```python
import os
os._exit(00)  # Полный перезапуск
```

### Groq API не отвечает
```
⚠️ Ошибка Groq: 401 Unauthorized
```
**Решение:** 
- Проверьте баланс токенов в аккаунте Groq
- Убедитесь, что ключ API правильный
- Проверьте, что выбранная модель доступна

### Неправильные ссылки
Если ссылки ведут на случайные товары, а не на идеи:
1. Перезапустите ядро в Kaggle
2. Выполните все ячейки заново
3. Убедитесь, что в `rag_pipeline.py` используется `create_search_links_for_idea()`

### Датасет не загружается
```
FileNotFoundError: [Errno 2] No such file or directory
```
**Решение:** Проверьте путь в `DATA_PATH`:
```python
# Правильный путь после Add Data
DATA_PATH = "/kaggle/input/datasets/username/dataset-name/filename.csv"
```
